# Valentine's Quest

A charming interactive Valentine's Day quest game where players need to find three keys through different challenges to reveal a special Valentine's message.

## Getting Started

This is a simple web application that runs directly in your browser. No special setup or installation is required!

### Running Locally

Just open the `index.html` file in your web browser, and you're ready to go!

### Customization

To personalize the quest for your Valentine, you can modify the following variables in `script.js`:

1. `numberOfHearts` - Change the number of hearts displayed in the first stage
2. `secondStepWords` - Modify the array of words used in the word unscramble challenge
3. `quizQuestions` - Customize the questions and answers for the quiz section
4. `valentinText` - Change the final Valentine's message

Example:
```javascript
const numberOfHearts = 20; // Change this number
const secondStepWords = ["Your", "Custom", "Words", "Here"];
const valentinText = "Your special message here!";
```

### Hosting

While you can run this directly from your local machine, you can also easily host it online. One simple option is to use [yhub.net](https://yhub.net) - just upload all the files, and you're good to go!

## Project Structure

- `index.html` - Main HTML file
- `script.js` - Game logic and customizable variables
- `styles.css` - Main styling
- `fireworks.css` - Fireworks animation styles
